# Curve Editor

一个基于 Python 标准库 `tkinter` 的曲线绘制小工具。

## 功能

- 左键在画布中添加锚点。
- 选中锚点后可拖动移动位置。
- 每个选中锚点会显示左右对称的方向手柄，拖动任一端会同步镜像另一端。
- 通过“绘画模式”开关控制是否允许在空白处新增点。
- 通过“显示点”开关控制是否显示锚点。
- 支持选择曲线颜色。
- 支持撤销、清空和右键删除点。

## 运行环境

- Python 3.x
- 仅使用标准库，不需要安装第三方依赖

## 启动方式

在项目目录下运行：

```bash
python main.py
```

## 操作说明

- 左键空白处：在绘画模式开启时添加新点
- 左键点：选中并拖动锚点
- 左键方向手柄端点：拖动手柄，调整曲线弯曲方向
- 右键点：删除该点
- `Undo`：撤销最后一个点
- `Clear`：清空全部点
- `绘画模式`：控制是否允许新增点
- `显示点`：控制锚点是否显示
- `曲线颜色`：打开颜色选择器，修改曲线颜色
